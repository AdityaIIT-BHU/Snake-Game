Snake-Game

Canvas Snake Game 🐍

A lightweight, fully interactive browser game engineered using HTML and JavaScript. 

While simple on the surface, this project was built to practice core programming fundamentals within a browser environment, including complex state management, asynchronous game loops, and mathematical collision detection.

Technical Highlights
* Custom Rendering Loop: Utilizes for smooth, real-time DOM rendering and frame-rate control.
* Coordinate-Based Logic: Implements a 2D grid system for array manipulation, handling the snake's dynamic queue-like movement and obstacle avoidance.
* Dynamic Difficulty: Engine scales game speed automatically based on the user's score interval.
* Persistent State: Leverages browser to cache high scores across active sessions.
